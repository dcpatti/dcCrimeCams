Development Method

This app is built with data captured from the Government of the District of Columbia -- DC Open Data.

Approximately eight datasets were captured through a variety of methods including API calls, CSV downloads, and GeoJSON calls. The data sets were stored in a combination of SQLite and CSV files, then normalized, distilled and cleaned. The cleaned data files were then processed into visualizations using the following technologies before being published to Heroku.

A Google Docs-based Requirements document and various checklists were used to manage the process, and all code was stored in GitHub for easy, collaborative access.

SQLite

SQLite is a relational database management system.

This project uses SQLite to build the user-searchable PSA/Camera table. SQLite is also the source for the json-based API service.

Python and Flask

Python is a programming language frequently used for data analysis. Flask is a micro web framework.

Together they can power dynamic, interactive, data-driven websites.

Leaflet

Leaflet.js is the leading JavaScript library for mobile-friendly, responsive maps.

This project uses D3 for the dynamic map.

D3

D3.js is a JavaScript library for producing dynamic, interactive data visualizations in web browsers.

This project uses D3 for the dynamic map and the interactive stack chart.

Slick.js

Slick.js is a JavaScript library for producing smooth-scrolling image carousels.

It was used to build the carousel on the main landing page. It's lightweight, fast and easy to implement.

Bootstrap

Bootstrap is a framework for developing responsive, mobile-first websites.

This project uses Bootstrap for layout, navigation, and style.
